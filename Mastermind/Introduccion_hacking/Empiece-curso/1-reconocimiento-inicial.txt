@@@ Fase de reconocimiento inicial - Enumeración de puertos con nmap

    1) Para empezar, probaremos con la IP, que nos ofrece el comando:
        route -n
    2) Comprobamos si la maquina que vamos a atacar se encuentra activa:
        ping -c IP
    Primero tener en cuenta el "ttl", esto nos indicará que sistema operativo tiene la maquina que vamos a atacar.
        64 == Linux
        128 == Windows
        Solaris/Aix == 254
    
    3) Ahora detectaremos con "nmap" que puertos estan abiertos por TCP:
    
    Para empezar a trabajar lo primero es crearnos una carpeta de trabajo, con una estructura de carpetas dentro.
    Para ello escribiremos un script que nos creara las carpetas automaticamente.
    El script es el siguiente:

        mkdir {nmap,content,scripts,tmp,exploits}
    
    Una ved echo esto, nos metemos en el directorio "nmap" y empezamos nuestra etapa de reconocimiento.

    4) Lo primero es escanear todo el rango de puertos:
        nmap ip -p- --open -T5 -v -n -oG allports
        -p- = Todos los puertos
        --open = filtrar puertos abiertos
        -T = Rango de velocidad
        -n = Sin resolución de DNS
        -oG = Exportar en formato grepeable
