Nos permite realizar un sin-fin de busquedas mas grande de lo habitual:

    COMANDOS:

        show modules == Ver que categorias de modulos hay

        modules search == Ver que modulos tenemos instalados

        marketplace search == Buscar módulos

        marketplace install nombre-module == Instalacion de modulo 

        modules load nombre-de-módulo 

            options list == Las opciones como valores que necesita

        db insert domains dominio.com == Guardar en la base de datos el dominio con el que trabajar
