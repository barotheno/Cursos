Comando que nos ayuda a volver en el tiempo. Pero no como "git checkout" que nos deja ir, mirar, pasear y volver. En casa bolvemos al pasado
sin la posibilidad de volver al futuro. Borramos y sobreescribimos.

    git reset --soft = Borramos todo el historial y los registros de GIT pero guardamos los cambios que tengamos en Staging, asi
                        podremos aplicar las últimas actualizaciones a un nuevo commit.

    git reset --hard = Borra todo, todo.
